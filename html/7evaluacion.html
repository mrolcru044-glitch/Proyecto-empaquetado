7.Evaluación.

Instrumentos de evaluación.

Para evaluar el desarrollo del proyecto se utilizarán diferentes instrumentos que permitirán valorar tanto el proceso como el producto final del alumnado.

Rúbrica del producto final.

Esta rúbrica permite evaluar aspectos como el empaquetado, el uso de materiales, el etiquetado, la presentación y el trabajo en equipo.

Lista de cotejo

Se utilizará durante las sesiones prácticas para observar el desempeño del alumnado.

Cuestionario digital

Se utilizarán herramientas digitales como Kahoot o Google Forms para evaluar los conocimientos teóricos del alumnado.

Evaluación del proceso

La evaluación será continua, teniendo en cuenta el trabajo diario, la participación y el progreso del alumnado.
